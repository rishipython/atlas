We need to implement faster pairwise squared Euclidean distances. Standard approach: compute Gram matrix. D[i,j] = ||x_i - x_j||^2 = ||x_i||^2 + ||x_j||^2 - 2 x_i·x_j. So we can compute row norms squared, outer sum, subtract 2 dot product.

Implementation:

def pairwise_sq_dist(X):
    X = np.asarray(X, dtype=np.float64)
    norms = np.sum(X**2, axis=1)
    D = norms[:, None] + norms[None, :] - 2 * X @ X.T
    return D
