def conv1d(x,h):
    x = np.asarray(x, dtype=np.float64)
    h = np.asarray(h, dtype=np.float64)
    N = x.shape[0]
    M = h.shape[0]
    if N==0 or M==0:
        return np.array([], dtype=np.float64)
    L = 1 << ((N+M-1-1).bit_length())  # next power of 2
    # compute rfft
    X = np.fft.rfft(x, n=L)
    H = np.fft.rfft(h, n=L)
    Y = X * H
    y = np.fft.irfft(Y, n=L)
    return y[:N+M-1]

Potential numeric differences: rfft/irfft might give slightly different results but should be close.

We can also use np.fft.fft with full complex arrays; that might be simpler. But rfft is more efficient.

Edge: if N+M-1 is odd? rfft length n returns n//2+1 frequencies. For L=next power of two, no issue.
