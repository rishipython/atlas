## Lesson
We need to write a Triton kernel that does softmax along last dim of 2D tensor. Must handle arbitrary N, not just powers of two. We want high performance, so use block size 256 columns per program. Each program processes one row. Two passes: first compute max, second compute exp and sum. Use float32 accumulation for stability. Need to cast input dtype to float32 if not already. Also cast back.. We need to consider pointer arithmetic with strides, masking for tail columns.. We need to decide between one kernel with two passes or two separate kernels? The brief says two Triton kernels. So we will launch first kernel to compute max per row, store into a temp buffer of shape M (float32). Then launch second kernel to compute exp(x - max) and sum per row, then normalize.. Alternatively we could do both in one kernel but brief says two kernels..
