We need to replace the slow triple‑loop with a fully vectorised routine that still shows off both SciPy and Numba usage while keeping everything numerically stable and fast enough for large matrices.
Let’s sketch the idea first:

- Compute each row’s squared norm `||x_i||²`.  
  The classic trick is  
  `D[i,j] = ||x_i||² + ||x_j||² – 2 * x_i·x_j`.  
  This gives us an `(N,N)` matrix with just one dense matrix multiplication plus some broadcasting.
- For the dense multiplication `X @ X.T` we’ll hand it off to SciPy’s BLAS wrapper (`scipy.linalg.blas.dgemm`).  
  That’s a thin layer over highly optimised LAPACK/BLAS routines.
- To satisfy the “uses Numba” rule we’ll wrap the row‑norm computation in a tiny `@njit` function with explicit double‑loop over rows & columns – it’s still compiled C‑level code so there’s no Python overhead.
- After assembling `D`, clip any tiny negatives caused by floating‑point round‑off (`np.maximum(... , 0.)`) and force the diagonal back exactly to zero – this keeps the output clean without affecting numerical tolerance.
- All operations stay on `float64`, matching the input type expected by callers.
- No pure Python loops remain at runtime – everything is either vectorised or compiled by Numba/BLAS.
- The final code stays under ~1200 characters while meeting every hard rule.



```python
import numpy as np
from scipy.linalg import blas
from numba import njit


@njit
def _row_norm_squared(x):
    n, d = x.shape
    out = np.empty(n)
    for i in range(n):
        s = 0.0
        for k in range(d):
            v = x[i, k]
            s += v * v
        out[i] = s
    return out


def pairwise_sq_dist(X: np.ndarray) -> np.ndarray:
    """Return the full pairwise squared Euclidean distance matrix."""
    # Compute squared norms per row using a tiny Numba‑compiled loop.
    norm_sq = _row_norm_squared(X)

    # Dot product via SciPy BLAS – fast dense multiplication.
    prod = blas.dgemm(alpha=1.0,
                      a=X,
                      b=X.T,
                      trans_a=False,
                      trans_b=True)

    # Assemble distance matrix using broadcasting.
    D = norm_sq[:, None] + norm_sq[None, :] - 2 * prod

    # Numerical safety:
    # Clip tiny negatives from round‑off errors and enforce exact zeros on diag.
    np.maximum(D, 0., out=D)
    idx = np.arange(D.shape[0])
    D[idx, idx] = 0.0

    return D
```
