#### Handling edge cases

- **Angle normalization** – SciPy handles any float angle; negative angles work fine.
- **Input shape** – If someone passes a 1‑D array or scalar by mistake, `rotate` will raise an error; I’ll guard against that by simply returning the input unchanged when its dimensionality is < 2.
- **Data type preservation** – If the input is integer typed (e.g., uint8), rotating with bilinear interpolation will produce floats (`float64`). I’ll cast back to the original dtype so that downstream code sees exactly what it expects.
- **Performance** – Setting `reshape=False` keeps output size equal to input size which avoids extra memory allocation for padding.
- **Interpolation order** – Order 1 (bilinear) gives a good trade‑off between quality and speed; higher orders would slow things down without much perceptible gain for typical test data.
- **Boundary handling** – Use `mode='constant'` with `cval=0` so pixels outside are zeroed out – matches typical behaviour of many rotation utilities.

```python
import random
from typing import Any, Dict
import numpy as np
import scipy.ndimage

# Seed just in case some downstream logic relies on randomness
random.seed(42)

def run_solver(problem):
    """
    Fast rotation solver using scipy.ndimage.rotate.
    
    Parameters
    ----------
    problem : dict
        Expected keys:
            'image' : np.ndarray  # 2-D or higher dimensional image data
            'angle' : float      # Rotation angle in degrees
    
    Returns
    -------
    np.ndarray
        Rotated image data with same shape as input.
    """
    img = problem['image']
    
    # Guard against degenerate shapes – nothing to rotate
    if img.ndim < 2:
        return img
    
    angle = float(problem.get('angle', 0))
    
    # Perform rotation
    rotated = scipy.ndimage.rotate(
        img,
        angle,
        reshape=False,
        order=1,
        mode='constant',
        cval=0.0,
        prefilter=False   # skip prefiltering for speed when possible
    )
    
    # Preserve original dtype if possible
    if rotated.dtype != img.dtype:
        try:
            rotated = rotated.astype(img.dtype)
        except Exception:
            pass
    
    return rotated
```
