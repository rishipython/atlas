```
def solve(problem):
    msg = problem.get("message", "")
    key_str = problem.get("key", "")
    # Convert key string possibly hex
    try:
        key_bytes = bytes.fromhex(key_str)
    except ValueError:
        key_bytes = key_str.encode()
    # optional salt
    salt = problem.get("salt")
    if salt is not None:
        # ensure salt as int or str
        if isinstance(salt,int):
            msg_bytes = (str(salt) + msg).encode()
        else:
            msg_bytes = (salt + msg).encode()
    else:
        msg_bytes = msg.encode()
    encoded_bytes = base64.b64encode(msg_bytes)
    encoded_str = encoded_bytes.decode()
    # Compute HMAC
    import hashlib
    h = hmac.new(key_bytes, encoded_str.encode(), hashlib.sha256).hexdigest()
    return {"encoded": encoded_str, "hmac": h}
```

* **Non‑ASCII characters** – We always UTF‑8‑encode the message before Base‑64ing it so Unicode works out of the box.
* **Key format** – The harness might supply the key either as raw ASCII text or as a hex dump (`"a1b2c3..."`). Trying `bytes.fromhex` first gives us a clean binary blob when possible; if it fails we fall back on plain ASCII encoding.
