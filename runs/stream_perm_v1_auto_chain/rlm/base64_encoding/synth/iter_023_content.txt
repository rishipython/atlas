```python
import base64
import hmac
import os
from typing import Any, Dict

class Base64Encoding:
    def solve(self, problem: Dict[str, Any]) -> str:
        # Normalise payload
        payload = problem.get("data", b"")
        if isinstance(payload, str):
            payload = payload.encode()

        # Get or generate key
        key = problem.get("key")
        if key is None:
            key = os.urandom(32)
        elif isinstance(key, str):
            key = key.encode()

        # Compute HMAC digest
        digest = hmac.new(key, payload, digestmod="sha256").digest()

        # Base‑64 encode
        encoded = base64.b64encode(digest).decode()

        return encoded

def run_solver(problem):
    return Base64Encoding().solve(problem)
```

This satisfies the “no Python‑level loops” rule because every heavy operation is delegated to compiled code (`hmac`, `base64`, `os`).  
It also keeps dependencies minimal – only the modules listed in the brief are used – so it should compile cleanly on any platform where those standard libraries exist.
