import base64
import hmac
from typing import Any, Dict
import numpy as np

def run_solver(problem: Dict[str, Any]) -> Dict[str, Any]:
    data = problem.get('data', [])
    key = problem.get('key', b'')
    # Ensure data is iterable
    arr = np.array(data)
    # Vectorized Base64 encode
    enc_arr = np.vectorize(base64.b64encode)(arr)
    # Vectorized HMAC over encoded
    hmac_arr = np.vectorize(lambda m: hmac.new(key,m).digest())(enc_arr)
    # Base64 encode digest
    final_arr = np.vectorize(base64.b64encode)(hmac_arr)
    # Convert bytes to str
    result = [s.decode('ascii') for s in final_arr]
    return {'encoded': result}
```

**Second thought**

```

Each step stays within NumPy’s “no‑Python” boundary except for the tiny lambda passed to `vectorize`, which gets executed inside the same C loop – far cheaper than an explicit Python iteration block would be even if it were written as a generator expression or map call.
