We also need to consider negative shifts: cross‑correlation result is centered; the peak index relative to center gives shift sign. We'll compute offset = peak_index - (shape-1). For 2D, we get two offsets.

Let's sketch the algorithm:

```
def solve(problem):
    img1 = jnp.array(problem['image1'])
    img2 = jnp.array(problem['image2'])
    # Ensure same shape
    shape = jnp.maximum(img1.shape, img2.shape)
    # Pad both images
    pad1 = ((0, shape[0]-img1.shape[0]), (0, shape[1]-img1.shape[1]))
    pad2 = ((0, shape[0]-img2.shape[0]), (0, shape[1]-img2.shape[1]))
    img1_p = jnp.pad(img1, pad1)
    img2_p = jnp.pad(img2, pad2)
    # Compute FFTs
    f1 = jnp.fft.rfftn(img1_p)
    f2 = jnp.fft.rfftn(img2_p)
    # Cross-correlation via inverse FFT of product of conj(f1) * f2
    corr = jnp.real(jnp.fft.irfftn(jnp.conj(f1) * f2))
    # Find peak
    idx = jnp.unravel_index(jnp.argmax(corr), corr.shape)
    # Compute shift relative to center
    center = tuple(s//2 for s in corr.shape)
    shift_y = idx[0] - center[0]
    shift_x = idx[1] - center[1]
    # Apply shift using scipy.ndimage.shift
    shifted_img2 = scipy.ndimage.shift(problem['image2'], shift=(shift_y, shift_x), order=3)
    return shifted_img2
```
