```
def solve(self, problem):
    image = np.asarray(problem["image"], dtype=np.float64)
    matrix = np.asarray(problem["matrix"], dtype=np.float64)
    transformed = scipy.ndimage.affine_transform(image, matrix, order=self.order,
                                                 mode=self.mode)
    return {"transformed_image": transformed}
```

This satisfies constraints and likely faster due to single conversion step instead of repeated conversions inside function calls for each pixel maybe?
