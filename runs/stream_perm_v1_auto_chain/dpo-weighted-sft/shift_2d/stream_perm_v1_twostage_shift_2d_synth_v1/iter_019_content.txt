```
def fft_cross_correlation(a,b):
    # Pad both arrays to shape sum -1
    shape = tuple(s_a + s_b - 1 for s_a,s_b in zip(a.shape,b.shape))
    A = np.zeros(shape,dtype=np.complex128)
    B = np.zeros(shape,dtype=np.complex128)
    A[:a.shape[0], :a.shape[1]] = a
    B[:b.shape[0], :b.shape[1]] = b
    FA = np.fft.rfftn(A)
    FB = np.fft.rfftn(B)
    corr = np.real(np.fft.irfftn(FA * FB.conj(), shape))
    return corr
```

This uses rfftn/irfftn for real input efficiency and avoids importing signal module entirely while staying within NumPy functions which are allowed since imported already via numpy alias.
