X = problem['X']  # shape (N,2)
Y = problem['Y']  # shape (N,2)
centroid_X = X.mean(axis=0)
centroid_Y = Y.mean(axis=0)
Xc = X - centroid_X
Yc = Y - centroid_Y
C = Xc.T @ Yc  # shape (2,2)
U,S,Vt = np.linalg.svd(C)
R = Vt.T @ U.T
if np.linalg.det(R) < 0:
    Vt[-1,:] *= -1
    R = Vt.T @ U.T
theta = np.arctan2(R[1,0], R[0,0])  # angle
return {'rotation': R, 'angle': theta}

Let’s assume the input `problem` dictionary contains two key‑value pairs – `'X'` and `'Y'` – each an \(N\times 2\) array of coordinates that differ only by an unknown planar rotation (no scaling or reflection).  
The classic way to recover the optimal rotation is a Procrustes alignment:

1. **Center** both point sets by subtracting their centroids – this removes translation.
2. Compute the cross‑covariance matrix  
   \(C = X_{\text{centered}}^{\top} Y_{\text{centered}}\) – a \(2\times 2\) matrix.
3. Perform an SVD on \(C\): \(C = U \Sigma V^{\top}\).
4. The optimal rotation is \(R = V U^{\top}\).  
   If \(\det(R)<0\), flip the sign of the last column of \(V\) before recomputing \(R\) so that \(\det(R)=+1\).
5. Extract the rotation angle from \(R\) via `atan2`.
